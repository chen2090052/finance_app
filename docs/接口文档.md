## 1. 流水列表

| 项目 | 内容 |
|------|------|
| 接口名称 | 流水列表 |
| 接口路径 | / |
| 请求方式 | GET |

### 请求参数

| 参数 | 类型 | 必传 | 说明 |
|------|------|------|------|
| year | Integer | 否 | 年份，默认当前年 |
| month | Integer | 否 | 月份(1-12)，默认当前月 |
| type | String | 否 | 筛选类型：INCOME(收入) / EXPENSE(支出) |
| category | Integer | 否 | 筛选分类ID |

### 返回参数

返回 HTML 页面，包含：

| 字段 | 类型 | 说明 |
|------|------|------|
| transactions | Array | 流水记录列表 |
| stats.income | Decimal | 月收入汇总 |
| stats.expense | Decimal | 月支出汇总 |
| stats.balance | Decimal | 月结余 |
| year | Integer | 当前年份 |
| month | Integer | 当前月份 |

| 接口名称 | 新增流水 |
| 接口路径 | /create/ |
| 请求方式 | GET / POST |

| 参数 | 类型 | 必传 | 说明 |
|------|------|------|------|

## 8. 删除分类

| 项目 | 内容 |
|------|------|
| 接口名称 | 删除分类 |
| 接口路径 | /categories/{id}/delete/ |
| 请求方式 | POST |

### 错误码

| 状态 | 说明 |
|------|------|
| 成功 | 跳转回分类列表，显示成功消息 |
| 失败 | 分类下有流水记录时禁止删除，显示错误消息 |

## 9. 统计报表

| 项目 | 内容 |
|------|------|
| 接口名称 | 统计报表 |
| 接口路径 | /statistics/ |
| 请求方式 | GET |

### 请求参数

| 参数 | 类型 | 必传 | 说明 |
|------|------|------|------|
| year | Integer | 否 | 年份，默认当前年 |
| month | Integer | 否 | 月份(1-12)，默认当前月 |

### 返回数据

| 字段 | 类型 | 说明 |
|------|------|------|
| stats.income | Decimal | 月收入 |
| stats.expense | Decimal | 月支出 |
| stats.balance | Decimal | 月结余 |
| stats.income_by_cat | Array | 各收入分类汇总 |
| stats.expense_by_cat | Array | 各支出分类汇总 |
| annual_income | Decimal | 年总收入 |
| annual_expense | Decimal | 年总支出 |
| annual_balance | Decimal | 年总结余 |

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 表单验证失败 | 返回表单页，字段旁显示红色错误提示 |
| 404 | 数据不存在或已删除 |
| 403 | CSRF 验证失败 |
