## 1. 流水列表

| 项目 | 内容 |
|------|------|
| 接口名称 | 流水列表 |
| 接口路径 | / |
| 请求方式 | GET |

### 请求参数

| 参数 | 类型 | 必传 | 说明 |
|------|------|------|------|
| year | Integer | 否 | 年份，默认当前年 |
| month | Integer | 否 | 月份(1-12)，默认当前月 |
| type | String | 否 | 筛选类型：INCOME(收入) / EXPENSE(支出) |
| category | Integer | 否 | 筛选分类ID |

### 返回参数

返回 HTML 页面，包含：

| 字段 | 类型 | 说明 |
|------|------|------|
| transactions | Array | 流水记录列表 |
| stats.income | Decimal | 月收入汇总 |
| stats.expense | Decimal | 月支出汇总 |
| stats.balance | Decimal | 月结余 |
| year | Integer | 当前年份 |
| month | Integer | 当前月份 |

## 11. 用户登录

| 项目 | 内容 |
|------|------|
| 接口名称 | 用户登录 |
| 接口路径 | /accounts/login/ |
| 请求方式 | GET / POST |

### POST 请求参数

| 参数 | 类型 | 必传 | 说明 |
|------|------|------|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码 |

### 返回说明

- 成功：跳转至流水列表页（或 `?next=` 指定页面）
- 失败：返回登录页，显示"用户名或密码错误"消息

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 表单验证失败 | 返回表单页，字段旁显示红色错误提示 |
| 401 | 未登录，跳转到登录页 |
| 403 | 无权限（非管理员访问管理端）/ CSRF 验证失败 |
| 404 | 数据不存在或已删除 |
