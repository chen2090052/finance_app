# 个人记账本 - 需求文档

> 版本：v1.0
> 日期：2026-06-29

## 一、项目概述

一个纯 Django + Python 的个人记账本 Web 应用，方便记录日常收支、管理分类、查看统计报表。

## 二、功能需求

### 2.1 流水管理
- 新增流水：记录一笔收支（类型、分类、金额、日期、备注）
- 编辑流水：修改已有流水记录
- 删除流水：软删除流水记录
- 流水列表：按日期倒序展示，支持按年/月/类型/分类筛选
- 月度统计卡片：列表顶部展示当月收入、支出、结余汇总

### 2.2 分类管理
- 新增分类：设置分类名称、类型（收入/支出）、图标
- 编辑分类：修改分类信息
- 删除分类：删除时校验是否有流水关联，有则禁止删除
- 分类按收入/支出分区展示

### 2.3 统计报表
- 月度统计：收入/支出/结余汇总、按分类的收入/支出明细
- 年度汇总：全年总收入/总支出/总结余
- 支持按年/月切换查看

## 三、数据模型

### 3.1 分类表 (Category)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| name | VARCHAR(50) | 分类名称 |
| type | VARCHAR(10) | 类型：INCOME=收入, EXPENSE=支出 |
| icon | VARCHAR(10) | 图标 emoji |
| sort_order | INT | 排序 |
| is_deleted | BOOLEAN | 软删除标记 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 3.2 流水表 (Transaction)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| amount | DECIMAL(12,2) | 金额 |
| type | VARCHAR(10) | 类型：INCOME=收入, EXPENSE=支出 |
| category_id | BIGINT FK | 关联分类 |
| note | VARCHAR(200) | 备注 |
| date | DATE | 日期 |
| is_deleted | BOOLEAN | 软删除标记 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

## 四、接口设计

| 路径 | 方法 | 说明 |
|------|------|------|
| / | GET | 流水列表+月度统计 |
| /create/ | GET/POST | 新增流水 |
| /{id}/edit/ | GET/POST | 编辑流水 |
| /{id}/delete/ | POST | 删除流水 |
| /categories/ | GET | 分类管理 |
| /categories/create/ | GET/POST | 新增分类 |
| /categories/{id}/edit/ | GET/POST | 编辑分类 |
| /categories/{id}/delete/ | POST | 删除分类 |
| /statistics/ | GET | 统计报表 |
| /load-categories/ | GET | AJAX加载分类选项 |

## 五、页面设计

| 页面 | 说明 |
|------|------|
| 流水列表页 | 顶部月度统计卡片 + 筛选条件 + 流水表格 |
| 新增/编辑流水页 | 表单：类型、分类、金额、日期、备注 |
| 分类管理页 | 收入分区 + 支出分区，各含列表 |
| 新增/编辑分类页 | 表单：名称、类型、图标、排序 |
| 统计报表页 | 月度汇总卡片 + 按分类明细 + 年度汇总 |

## 六、用户认证

### 6.1 功能需求
- 用户注册：用户名、邮箱、密码，注册后跳转到登录页
- 用户登录：用户名 + 密码验证，登录后跳转到流水列表
- 用户退出：清除会话，跳转到登录页
- 权限控制：未登录用户只能访问登录页和注册页
- 角色区分：普通用户和管理员（is_staff）

### 6.2 角色权限

| 角色 | 权限范围 |
|------|---------|
| 未登录 | 仅可访问登录页、注册页 |
| 普通用户 | 管理自己的流水记录（增删改查）、查看统计报表 |
| 管理员 | 普通用户所有权限 + 分类管理（增删改查）+ 管理后台 + 用户管理 + Django Admin |

## 七、管理后台

### 7.1 功能需求
- 数据总览仪表盘：用户数、流水总数、总收入、总支出、最近流水
- 用户管理：查看用户列表、新增用户、编辑用户（启用/禁用、设置管理员权限）
- 快捷入口：用户管理、流水管理、分类管理、Django Admin 链接
